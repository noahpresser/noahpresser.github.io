Pocket Capture Calculator


Pocket Capture Calculator is an application I built, using Unity, for Pokémon gamers. Using the app, a player can determine his probability of capturing a Pokémon. It currently functions for all games in generation I through VI (Omega Ruby and Alpha Sapphire, for those of you counting).

Virtual Reality - Light Museum


Using VR, our group simulated artwork created by James Turrell. James Turrell's work involves manipulating light to display various illusions. This project is unique because, aside from viewing it in VR, Turrell's work is very fleeting. The exhibit is seasonal and seen only in a few museums worldwide.
